Overview

Jena AI is a toolkit and runtime for building AI-powered VTubers and virtual assistants. It combines input handling, agent orchestration, multimodal outputs (voice, animation, overlays), and flexible integration points so creators can build interactive characters.

Installation

git clone https://github.com/Saidie000/JenaAI-VTuber.git
cd JenaAI-VTuber
npm install

Make sure the logo is present at /docs/assets/logo.svg. The branch chore/logo-svg includes an SVG wrapper that references the raster logo.

Usage

See the repository README and the docs/ directory for examples and CLI instructions. To preview these files locally, run a simple static server from the repository root and open /docs/index.html.

Architecture

  • Avatar rendering (Three.js)
  • Virtual input and game controllers
  • Modular hot-swappable modules via WebSocket
  • AI integration (WebLLM or chosen LLM)
